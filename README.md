# 💘 Girlfriend Finder: Moscow "Fun Temperament" Strategy

Автоматизированный пайплайн для поиска, глубокого психометрического анализа и первичного контакта с потенциальными партнершами ВКонтакте в Москве. 

Проект реализует научно обоснованную стратегию поиска партнера с **«веселым нравом»**, используя модель **Big Five (OCEAN)** и теорию **стилей юмора**. Вместо поверхностной оценки по фото, система анализирует интересы, подписки, статус и тексты профиля, чтобы выявить глубинные черты личности: экстраверсию, эмоциональную стабильность и тип юмора.

## 🧠 Научная методология

В основе алгоритма лежит деконструкция понятия «веселый нрав» на измеримые психологические метрики.

### 1. Модель Big Five (OCEAN)
Система оценивает пять ключевых параметров личности по шкале от 0 до 100:

| Параметр | Что ищем | Почему это важно |
| :--- | :--- | :--- |
| 🟢 **Экстраверсия** | Высокий уровень (>60) | Источник энергии в общении, жизнерадостность, активность. Экстраверты чаще инициируют контакт и создают атмосферу праздника. |
| 🔵 **Нейротизм** | **Низкий** уровень (<40) | Эмоциональная стабильность. Низкий нейротизм означает отсутствие склонности к тревоге, драмам и меланхолии. Критически важно для долгосрочного счастья. |
| 🟠 **Добросовестность** | Средний/Высокий | Надежность, ответственность, умение доводить дела до конца. Гарантирует предсказуемость и безопасность в отношениях. |
| 🟣 **Открытость опыту** | Высокий уровень | Любопытство, креативность, готовность к путешествиям и новым впечатлениям. Делает отношения насыщенными и нескучными. |
| 🟡 **Доброжелательность** | Высокий уровень | Эмпатия, доверие, стремление к сотрудничеству. Снижает вероятность конфликтов. |

### 2. Типы Юмора (Humor Styles)
Юмор — не просто развлечение, а механизм адаптации. Алгоритм классифицирует стиль юмора пользователя:

*   ✅ **Аффилиативный (Affiliative):** Шутки для сближения, создания атмосферы, поддержки группы. *Позитивный маркер.*
*   ✅ **Самоирония (Self-enhancing):** Способность смеяться над собой, устойчивость к трудностям через юмор. *Сильнейший позитивный маркер.*
*   ❌ **Агрессивный (Aggressive):** Сарказм, насмешки, унижение других под видом шутки. *Красный флаг (Red Flag).*
*   ❌ **Саморазрушающий (Self-defeating):** Шутки за свой счет ради одобрения других. *Негативный маркер.*

### 3. Генерация персонализированного сообщения
Для каждого кандидата с высоким скором совместимости LLM генерирует уникальное сообщение («Icebreaker»), которое:
*   Использует имя девушки.
*   Опирается на её конкретные интересы (из раздела "Подписки" или "О себе").
*   Соответствует её психотипу (например, более энергичное для экстравертов, более мягкое для интровертов).
*   Написано живым языком, без шаблонных фраз «привет, как дела».

## 🛠 Технологический стек

*   **Язык:** Python 3.10+
*   **Сбор данных:** `vk_api` (официальный API ВКонтакте).
*   **Искусственный интеллект:** Локальная LLM через **Ollama**.
    *   *Модели:* `qwen2.5:1.5b` (рекомендуемая), `llama3.2:3b`, `phi3:mini`.
    *   *Преимущества:* Полная приватность данных, отсутствие лимитов API, бесплатно, работает офлайн.
*   **Обработка:** Последовательный анализ (CPU-optimized) с автосохранением после каждого профиля.
*   **Отчетность:** Генерация интерактивного HTML-отчета с кнопками копирования.
