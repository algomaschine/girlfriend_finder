# 🚀 Girlfriend Finder: Moscow "Fun Temperament" Strategy

Автоматизированный пайплайн для поиска и психометрического анализа потенциальных партнерш ВКонтакте в Москве. 
Проект использует научно обоснованный подход на основе модели **Big Five (OCEAN)** и теории **стилей юмора** для выявления девушек с «веселым нравом» (высокая экстраверсия, низкий нейротизм, адаптивный юмор).

## 🧠 Научная основа анализа

Скрипт оценивает профили не по внешности, а по психологическим маркерам, соответствующим запросу на «веселый нрав»:

1.  **Big Five (OCEAN):**
    *   🟢 **Экстраверсия (Extraversion):** Поиск энергичных, общительных, жизнерадостных личностей.
    *   🔵 **Нейротизм (Neuroticism):** Фильтрация эмоционально нестабильных; приоритет спокойным и позитивным.
    *   🟠 **Добросовестность (Conscientiousness):** Оценка надежности и ответственности.
    *   🟣 **Открытость опыту (Openness):** Поиск творческих личностей, готовых к приключениям.
    *   🟡 **Доброжелательность (Agreeableness):** Способность к эмпатии и сотрудничеству.

2.  **Стили юмора (Humor Styles):**
    *   ✅ **Аффилиативный:** Юмор для сближения и создания атмосферы.
    *   ✅ **Самоирония:** Здоровое отношение к себе и трудностям.
    *   ❌ **Агрессивный/Саркастичный:** Маркеры токсичности (понижают скор).

3.  **Генерация сообщения:**
    Для каждого подходящего кандидата ИИ генерирует уникальное, «человеческое» приветственное сообщение, основанное на её интересах и психотипе, чтобы максимизировать шанс ответа.

## 🛠 Технологии

*   **Язык:** Python 3.10+
*   **VK API:** Сбор данных (профили, подписки, стены).
*   **LLM (Local AI):** [Ollama](https://ollama.com/) с моделями `qwen2.5:1.5b` или `llama3.2:3b`.
    *   *Почему локально?* Полная приватность, отсутствие лимитов API, бесплатно, работает без интернета после загрузки модели.
*   **Конкурентность:** Асинхронный сбор данных + последовательный анализ (для стабильности CPU).
*   **Отчетность:** Генерация Markdown-отчетов и JSON-баз данных.
